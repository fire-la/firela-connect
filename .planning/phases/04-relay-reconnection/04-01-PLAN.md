- **Missing**: No jitter, causing synchronized reconnection storms (thundering herd)

# Research Findings (AWS Architecture Blog - Full Jitter)
Full Jitter algorithm (recommended by AWS):
```
sleep = random(0, min(cap, base * 2 ** attempt))
```
- Spreads reconnection attempts evenly
- Reduces server load by 50%+ compared to no-jitter
- Prevents thundering herd when multiple clients disconnect simultaneously

```typescript
// Full Jitter: sleep = random(0, min(cap, base * 2 ** attempt))
export function calculateBackoffDelay(
  baseDelay: number,
  maxDelay: number,
  attempt: number
): number {
  const exponentialDelay = baseDelay * Math.pow(2, attempt)
  const cappedDelay = Math.min(maxDelay, exponentialDelay)
  return Math.random() * cappedDelay
}
```
